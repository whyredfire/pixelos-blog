# June 2026 Update

For this month's update, we've incorporated the June security patch from `android-security-16.0.0_r7` into Android 16 QPR2. Alongside that, we've also included a set of improvements and fixes focused on usability, polish, and everyday quality-of-life enhancements.

<Banner src="https://raw.githubusercontent.com/PixelOS-CI/blog_assets/refs/heads/main/2026-06-10/banner.webp" />

With Android 17 expected to be released later this month (following Google's platform release schedule), this will hopefully be the final Android 16-based PixelOS update before we begin the transition to the next Android version.

As always, detailed changelogs can be found on our [changelogs page](https://blog.pixelos.net/changelogs).

## This Month's Banner
A special shoutout to our community member [nexy](https://t.me/spesmynuts) for sharing this month's release banner with us. We liked it so much that we decided to feature it in this release post, marking the first time we've featured community-created artwork in a PixelOS announcement.
